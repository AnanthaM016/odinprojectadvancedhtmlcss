# Odin Project Advanced HTML & CSS

## Assignment: Personal Portfolio Website

A fully responsive personal portfolio website built using **HTML5** and **CSS3**.  
This project is designed to showcase projects and contact information with a modern, clean aesthetic.

---

## Features

- **Responsive Layout:** Adapts seamlessly to mobile, tablet, and desktop screens.
- **Modern Design:** Includes a distinctive angled/diagonal hero section and a card-based project grid.
- **Navigation:** Fixed navigation bar with smooth scrolling (`scroll-behavior: smooth`) for improved user experience.
- **Typography:** Uses **Playfair Display** for headings and **Roboto** for body text via Google Fonts.
- **Icons:** Scalable SVG icons for social media and contact details.

---

## Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Flexbox and CSS Grid for layout; CSS variables for color management.
